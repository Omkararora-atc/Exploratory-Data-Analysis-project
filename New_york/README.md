# 🗽 New York Airbnb Exploratory Data Analysis

Welcome to the **New York** folder of the [Exploratory Data Analysis Project](https://github.com/Omkararora-atc/Exploratory-Data-Analysis-project)! This directory contains a detailed data analysis of Airbnb listings in New York City.

---

## 📊 Project Overview

This analysis uses Airbnb data to uncover insights into:
- **Neighbourhood popularity**
- **Pricing trends**
- **Room type distribution**
- **Review patterns**
- **Availability by neighbourhood**

By leveraging data visualization and statistical techniques, we provide a comprehensive understanding of the short-term rental market in New York.

---

## 📁 Contents

- [`Analysis.ipynb`](./Analysis.ipynb): Jupyter notebook with step-by-step EDA, visualizations, and key findings.

---

## 🌆 Key Insights

- **Most Popular Neighbourhoods:**  
  Manhattan leads in number of listings, followed by Brooklyn and Queens.
- **Pricing & Listing Counts:**  
  Manhattan dominates across all price ranges; Brooklyn has significant listings in lower price segments.
- **Room Types:**  
  Entire home/apartment listings are the most common, especially in higher price brackets.
- **Top Neighbourhoods by Reviews & Availability:**  
  Williamsburg, Bedford-Stuyvesant, and Harlem stand out for both reviews and availability.

---
