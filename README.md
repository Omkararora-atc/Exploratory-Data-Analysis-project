# 📊 Exploratory Data Analysis Project

Welcome! This repo showcases two cool EDA projects, each in its own folder. Dive in and explore! 🚀

---

## 📂 Folder Structure

### 🗽 [`New_york`](./New_york)
Airbnb NYC data analysis:
- 📄 `AB_NYC_2019.csv`: Raw data
- 📑 `cleaned_new.csv`: Cleaned data
- 📓 `Analysis.ipynb`: Main analysis notebook
- 🧹 `cleaning_data.ipynb`: Data cleaning steps
- 📘 `README.md`: Project details

### 🍽️ [`Zomato EDA`](./Zomato%20EDA)
Zomato restaurant data analysis:
- 📄 `zomato.csv`: Raw data
- 🌍 `Country-Code.xlsx` / `Country-Code.zip`: Country info
- 🗃️ `zomato.zip`: Zipped dataset
- 📓 `Analysis.ipynb`: Main analysis notebook
- 📘 `README.md`: Project details

---

## 🚦 Get Started

- Open a folder to explore its notebooks and data.
- Run `.ipynb` files in Jupyter for step-by-step EDA.
- Check each folder's README for more info!
